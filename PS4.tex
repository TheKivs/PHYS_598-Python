\section{PHYS 598 SDA RECITATION 4 - PROBLEM SET}

**(Starred problems to be done only after you're done with the unstarred problems.)**\\

\textbf{1.} Recall that a real-valued matrix $\sO$ is orthogonal if $\sO\sO^t = \mathbb{I}$. Also, a real symmetric matrix $\sR$ is defined by $R=R^t$. Suppose that you are given an invertible $n \times n$ matrix $M$.

\begin{itemize}
\item \emph{Polar decomposition}: Prove that $\sR = \sqrt{MM^t}$ is (positive) symmetric and $\sO = \sR^{-1}M$ is orthogonal. This is called the polar decomposition of $M$. (Why?)
\item $^*$\emph{Conformal matrices}: If $MM^t = \lambda \mathbb{I}$ for some $\lambda \in \mathbb{R}$. Determine the set of all $2\times 2$ conformal matrices. 

(In physics, orthogonal transformations appear everywhere, and symmetries of physical systems are described by groups $O(n)$, which have orthogonal matrices as a representation. In special relativity, the Lorentz group is \emph{isomorphic} to the orthogonal group $O(3,1)$ that mixes space and time. If you ever decide to read about AdS/CFT, conformal matrices are of prime relevance.)
\end{itemize}

\textbf{2.} Let $\Omega$ be a convex domain in $\mathbb{R}^n$. We consider the Hilbert space $L^2(\Omega)$ of square integrable functions with scalar product
\[\langle f,g\rangle = \int_\Omega f(x)g^*(x) dx.\]
\begin{itemize}
\item
Consider the Hilbert space $L^2[0, 1]$. Find a non-trivial function
\[f(x) = ax^3 + bx^2 + cx + d,\]
such that
\[\langle f(x),x\rangle = 0, ~~~~~~ \langle f(x),x^2\rangle = 0, ~~~~~~ \langle f(x),x^3\rangle = 0,\]
where $\langle \cdot,\cdot\rangle$ denotes the scalar product.
\item Consider the Hilbert space $L^2([-1, 1])$. The \emph{Chebyshev polynomials} are defined by
\[T_n(x):= \cos (n \cos^{-1}x), ~~~ n\in \mathbb{Z}^+.\]
Using a computer or otherwise, convince yourself that $T_n(x) \in L^2([-1, 1])$.

(Chebyshev polynomials are used in study of nonlinear systems, a notable example being the system of two independent oscillators which plot the \emph{Lissajous curves}.)
\end{itemize}

\textbf{3.} Can one construct an orthonormal basis in the Hilbert space $L^2(\mathbb R)$ starting from ($\sigma > 0$)
\[e^{-|x|/\sigma}, ~xe^{-|x|/\sigma}, ~ x^2e^{-|x|/\sigma}, \ldots ?\]

A note on notation : Typically, $\ell_p$ is used to indicate a $p$-summable discrete set of values. For example, $\ell_p(\mathbb Z^+)$ is the set of complex-valued sequences $\{a_n\}$ such that $\sum_{n\in \mathbb Z^+} |a_n|^p < \infty$. $L^p$ is usually used to indicate $p$-summable functions (with respect to some measure) on a non-discrete measure space, such as the usual $L^p(\mathbb R)$, the set of functions $f$: $\mathbb R \to \mathbb C$ such that $\int _\mathbb{R} |f(x)|^p dx<\infty.$ The main point is that they are mathematically different notation for the same concept. \\
