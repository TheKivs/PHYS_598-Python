\textbf{1.} Consider a $p$-dimensional vector $\bm{X} = \{X_1,\ldots,X_p\}$ and a $q$-dimensional vector $\bm{Y}= \{Y_1,\ldots,Y_q\}$. Then the \textbf{covariance matrix} $\Sigma(\bm X)$ is defined by
\begin{equation*}
\Sigma_{i,j}(\bm X) = \text{cov}(X_i,X_j).
\end{equation*}
Similarly, define the \emph{cross}-covariance matrix $\Sigma_{i,j}(\bm X,\bm Y) = \text{cov}(X_i,Y_j)$.
Your task is to convince yourself of the following properties of $\Sigma$:
\begin{enumerate}[(a)]
\item
$\Sigma(\bm X,\bm Y) = \Sigma(\bm Y, \bm X)^T$. (C'mon, obviously!)
\item Linearity, $\Sigma(\alpha \bm X, \bm Y) = \alpha\Sigma( \bm X, \bm Y)$, where $\alpha \in \mathbb{R}$, etc.
\item If $p = q$, then $\Sigma(\bm X + \bm Y) = \Sigma(\bm X) + \Sigma(\bm X,\bm Y) +  \Sigma(\bm Y,\bm X) + \Sigma(\bm Y)$. This should follow from (b).
\item For a $q\times p$ matrix $\bm A$ and a $q$-dim vector $\bm a$, $\Sigma(\bm{AX + a}) = \bm A \Sigma(\bm X) \bm A^T.$
\end{enumerate}

\textbf{3.} You are given a power grid, that is a square of side S ($\leq 10^9$). You are also given $N$ special points ($N$ $\leq$ 5000). Each of the $N$ special points has a different power $R_i$ $(\leq 10^9)$ associated with it. Each of these points can influence all points of the grid within a Manhattan distance of $R_i$ (if the Manhattan distance between the special point and any other point is $\leq$ $R_i$, it will be influenced). Find out the expected number of points that are influenced in the grid.\\
